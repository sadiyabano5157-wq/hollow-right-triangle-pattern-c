# ⭐ Hollow Right Triangle Pattern in C

## 📌 Description
This program prints a hollow right triangle star pattern using nested loops and conditional statements in C language.

## 💡 Pattern Output
Enter a row: 5
*
**
* *
*  *
*****

---

## 🧠 Concepts Used
- Nested loops
- Conditional statements
- Hollow pattern logic
- Row and column understanding

---

## ⚙️ Logic

Star is printed when:
- First column
- Diagonal side
- Last row

Otherwise spaces are printed.

- C Programming Language
